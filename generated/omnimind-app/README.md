## Local dev

```bash
# Backend
cd backend
python -m venv .venv
.venv\\Scripts\\activate   # Windows
pip install -r requirements.txt
uvicorn main:app --host 127.0.0.1 --port 8001 --reload

# Frontend (new terminal)
cd frontend
npm install
npm run dev
```
